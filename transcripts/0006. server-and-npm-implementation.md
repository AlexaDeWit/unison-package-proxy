# Server Logic and NPM Implementation

Re-implementation of the proxy server and NPM client using the
generalized `RegistryProxy` type system. Updates `RegistryClient`
to include the `Threads` ability (needed by the HTTP client library),
adds the generic proxy handler lifecycle, and provides a working
npm-specific implementation.

## Updated RegistryClient (adds Threads)

The HTTP client library requires the `Threads` ability for connection
pooling. This updates the `RegistryClient` record's IO fields to
include `Threads` alongside `IO` and `Exception`.

```unison
namespace RegistryProxy

{{ A registry client interface. Different registries (npm, PyPI, etc.)
provide different implementations of this record.

The proxy lifecycle using a {type RegistryClient}:

1. Call ''fetchMetadata'' or ''fetchArtifact'' on an upstream client.
2. Check ''statusCode'' of the {type RegistryResponse}.
3. If 2xx, forward ''responseBody'' to the caller.
4. If non-2xx, begin fallback with a different client.
5. On fallback success, use ''parsePackageInfo'' to extract metadata
   for the rules engine.
6. If the rules engine allows, forward the fallback response body.

The ''parse*'' functions are pure — they extract internal metadata
from a {type RegistryResponse} without performing IO. Each registry
implementation provides its own parsing logic (e.g., npm JSON codecs,
PyPI HTML index parsing). }}
type RegistryClient =
  { fetchMetadata :
      PackageId -> '{IO, Exception, Threads} RegistryResponse
  , fetchArtifact :
      PackageId -> Text -> '{IO, Exception, Threads} RegistryResponse
  , publishArtifact :
      PackageId -> Text -> Bytes
        -> '{IO, Exception, Threads} (Either Text ())
  , parsePackageInfo :
      RegistryResponse -> Either Text PackageInfo
  , parseVersionInfo :
      RegistryResponse -> Text -> Either Text VersionInfo
  , parseVersionList :
      RegistryResponse -> Either Text [Text]
  }

{{ Configuration for a single upstream registry, including the
client interface and any additional rules specific to this
upstream (beyond the global rules). }}
type Server.UpstreamConfig
  = { client : RegistryClient
    , additionalRules : RuleSet
    }

{{ Main proxy server configuration.

* ''privateUpstream'' — checked first; if it returns a successful
  response, served immediately without rules evaluation.
* ''publicUpstream'' — fallback when private misses; responses are
  subject to rules evaluation before serving.
* ''mirrorTarget'' — where allowed public packages are mirrored.
* ''globalRules'' — the deny-by-default rule set applied to all
  public upstream requests.
* ''listenPort'' — TCP port the proxy server listens on.

Per-upstream ''additionalRules'' are merged with ''globalRules''
via {mergeRuleSets} before evaluation. }}
type Server.ProxyConfig
  = { privateUpstream : Server.UpstreamConfig
    , publicUpstream : Server.UpstreamConfig
    , mirrorTarget : RegistryClient
    , globalRules : RuleSet
    , listenPort : Nat
    }

{{ Build the effective rule set for a given upstream by merging
the global rules with the upstream's additional rules. }}
Server.effectiveRules : RuleSet -> Server.UpstreamConfig -> RuleSet
Server.effectiveRules globalRules upstream =
  mergeRuleSets globalRules (Server.UpstreamConfig.additionalRules upstream)
```

## CVE Lookup Handler and Server Utilities

```unison
namespace RegistryProxy

{{ A no-op handler for {type CVELookup} that returns no CVEs
and reports no active vulnerabilities. Use this as a default
when CVE checking is not configured. }}
noCVELookup : '{g, CVELookup} a ->{g} a
noCVELookup thunk =
  h : Request {CVELookup} a -> a
  h = cases
    { a } -> a
    { CVELookup.lookupCVEs _ -> resume } ->
      handle resume [] with h
    { CVELookup.hasActiveCVE _ _ -> resume } ->
      handle resume false with h
  handle !thunk with h

{{ Extract the path from an {type HttpRequest} as a text string. }}
Server.requestPath : HttpRequest -> Text
Server.requestPath req =
  Path.toText (URI.path (HttpRequest.uri req))

{{ Convert a {type RegistryResponse} to an {type HttpResponse}
for serving to the client. Forwards the status code, content type,
and body bytes. }}
Server.registryResponseToHttp : RegistryResponse -> HttpResponse
Server.registryResponseToHttp resp =
  HttpResponse.HttpResponse
    (HttpResponse.Status.Status
      (RegistryResponse.statusCode resp) "OK")
    Version.http11
    (Headers.singleton "Content-Type"
      (RegistryResponse.contentType resp))
    (Body.Body (RegistryResponse.responseBody resp))

{{ Try fetching metadata from an upstream client. Returns
''Some'' response if the upstream returned a 2xx status,
''None'' otherwise. }}
Server.tryUpstream :
  RegistryClient
  -> PackageId
  ->{IO, Exception, Threads} Optional RegistryResponse
Server.tryUpstream client pkgId =
  resp = !(RegistryClient.fetchMetadata client pkgId)
  if isSuccess (RegistryResponse.statusCode resp) then
    Some resp
  else
    None
```

## Proxy Handler

```unison
namespace RegistryProxy

{{ Handle a request for a specific package through the public
upstream with rules evaluation. Called when the private upstream
does not have the package. }}
Server.handlePublicFallback :
  Server.ProxyConfig
  -> PackageId
  ->{IO, Exception, Threads} HttpResponse
Server.handlePublicFallback config pkgId =
  publicUpstream = Server.ProxyConfig.publicUpstream config
  publicClient = Server.UpstreamConfig.client publicUpstream
  match Server.tryUpstream publicClient pkgId with
    None ->
      Server.denyResponse
        ("package not found: " ++ PackageId.toText pkgId)
    Some resp ->
      match RegistryClient.parsePackageInfo publicClient resp with
        Left err ->
          Server.denyResponse ("metadata parse error: " ++ err)
        Right pkgInfo ->
          ctx = RuleContext.RuleContext pkgInfo None
          rules =
            Server.effectiveRules
              (Server.ProxyConfig.globalRules config)
              publicUpstream
          decision = noCVELookup '(evaluateRuleSet rules ctx)
          match decision with
            RuleDecision.Allow _ ->
              Server.registryResponseToHttp resp
            RuleDecision.Deny reason -> Server.denyResponse reason
            RuleDecision.NoDecision ->
              Server.denyResponse
                ("denied by default: " ++ PackageId.toText pkgId)

{{ Handle a request for a specific package through the proxy
lifecycle. Tries private upstream first, falls back to public
with rules evaluation. }}
Server.handlePackage :
  Server.ProxyConfig
  -> PackageId
  ->{IO, Exception, Threads} HttpResponse
Server.handlePackage config pkgId =
  privateClient =
    Server.UpstreamConfig.client
      (Server.ProxyConfig.privateUpstream config)
  match Server.tryUpstream privateClient pkgId with
    Some resp -> Server.registryResponseToHttp resp
    None -> Server.handlePublicFallback config pkgId

{{ The main proxy handler. Given a proxy configuration and a
registry-specific path-to-package-ID parser, handles an incoming
HTTP request through the proxy lifecycle:

1. Parse the request path into a {type PackageId}.
2. Try the private upstream — if it returns a 2xx response,
   serve it directly (no rules evaluation).
3. Fall back to the public upstream — if 2xx, parse metadata,
   evaluate rules, and serve or deny.
4. If both upstreams miss, return 404.

Deny decisions produce a 404 via {Server.denyResponse}. }}
Server.proxyHandler :
  Server.ProxyConfig
  -> (Text -> Optional PackageId)
  -> HttpRequest
  ->{IO, Exception, Threads} HttpResponse
Server.proxyHandler config parsePackageId req =
  path = Server.requestPath req
  match parsePackageId path with
    None -> Server.denyResponse ("unrecognized path: " ++ path)
    Some pkgId -> Server.handlePackage config pkgId
```

## Routing and Server Startup

```unison
namespace RegistryProxy

{{ Build HTTP {type Routes} for the proxy. Catches all requests
and dispatches to {Server.proxyHandler}. }}
Server.proxyRoutes :
  Server.ProxyConfig
  -> (Text -> Optional PackageId)
  -> Routes {IO, Exception, Threads}
Server.proxyRoutes config parsePackageId =
  handler = Handler cases
    req -> Server.proxyHandler config parsePackageId req
  Routes.default <<< handler

{{ Start the proxy server on the configured port. Blocks until
the server is stopped. }}
Server.startProxy :
  Server.ProxyConfig
  -> (Text -> Optional PackageId)
  ->{IO, Exception, Threads} ()
Server.startProxy config parsePackageId =
  port = Port.Port (Nat.toText (Server.ProxyConfig.listenPort config))
  serverConfig =
    server.Config.Config None port None None
  routes = Server.proxyRoutes config parsePackageId
  server.Config.serve routes serverConfig
```

## NPM Client Implementation

```unison
namespace RegistryProxy

{{ Parse an npm registry URL path into a {type PackageId}.

Handles both scoped and unscoped packages:
* ''/react'' → {PackageId.Unscoped} ''react''
* ''/@babel/core'' → {PackageId.Scoped} ''babel'' ''core''
* ''/@scope/name/-/name-1.0.0.tgz'' → {PackageId.Scoped} ''scope'' ''name''

Returns ''None'' for unrecognizable paths. }}
Npm.parsePackageId : Text -> Optional PackageId
Npm.parsePackageId path =
  stripped = Text.drop 1 path
  if Text.startsWith "@" stripped then
    match Text.split ?/ stripped with
      scopeWithAt +: rest -> match rest with
        name +: _ ->
          scope = Text.drop 1 scopeWithAt
          if scope == "" then None
          else if name == "" then None
          else Some (PackageId.Scoped scope name)
        _ -> None
      _ -> None
  else
    match Text.split ?/ stripped with
      name +: _ ->
        if name == "" then None
        else Some (PackageId.Unscoped name)
      _ -> None

{{ Make an HTTP GET request to an npm registry endpoint and
convert the response to a {type RegistryResponse}. Adds Bearer
token authentication if the endpoint's {type TokenProvider}
resolves to a token. }}
Npm.npmFetch :
  RegistryEndpoint -> Text -> '{IO, Exception, Threads} RegistryResponse
Npm.npmFetch endpoint urlPath = do
  url = RegistryEndpoint.endpointUrl endpoint ++ urlPath
  uri = URI.parse url
  baseReq = HttpRequest.get uri
  token = !(resolveToken (RegistryEndpoint.tokenProvider endpoint))
  req = match token with
    Some t ->
      HttpRequest.addHeader "Authorization" ("Bearer " ++ t) baseReq
    None -> baseReq
  httpResp = handle Http.request req with Http.handler
  sc = HttpResponse.Status.code (HttpResponse.status httpResp)
  bodyBytes = Body.toBytes (HttpResponse.body httpResp)
  ct =
    match Headers.getValues "content-type" (HttpResponse.headers httpResp) with
      v +: _ -> v
      [] -> "application/octet-stream"
  RegistryResponse.RegistryResponse sc [] bodyBytes ct

{{ Construct a {type RegistryClient} for an npm registry.
Uses the given {type RegistryEndpoint} for HTTP calls.

The ''parse*'' functions are stubs that return ''Left'' errors
until connected to the JSON codecs. }}
Npm.makeClient : RegistryEndpoint -> RegistryClient
Npm.makeClient endpoint =
  RegistryClient.RegistryClient
    (pkgId -> Npm.npmFetch endpoint ("/" ++ PackageId.toText pkgId))
    (pkgId -> ver ->
      Npm.npmFetch endpoint
        ("/" ++ PackageId.toText pkgId ++ "/-/" ++ ver))
    (pkgId -> ver -> bytes -> do Left "publish not yet implemented")
    (resp -> Left "parsePackageInfo not yet implemented")
    (resp -> ver -> Left "parseVersionInfo not yet implemented")
    (resp -> Left "parseVersionList not yet implemented")
```

namespace RegistryProxy

{{ Example proxy configuration for testing. Proxies from the
public npm registry with no private upstream and an AllowAll rule. }}
Npm.testConfig : Server.ProxyConfig
Npm.testConfig =
  publicEndpoint =
    RegistryEndpoint.RegistryEndpoint
      "https://registry.npmjs.org" TokenProvider.NoAuth
  publicClient = Npm.makeClient publicEndpoint
  dummyEndpoint =
    RegistryEndpoint.RegistryEndpoint
      "http://localhost:4873" TokenProvider.NoAuth
  dummyClient = Npm.makeClient dummyEndpoint
  Server.ProxyConfig.ProxyConfig
    (Server.UpstreamConfig.UpstreamConfig dummyClient emptyRuleSet)
    (Server.UpstreamConfig.UpstreamConfig publicClient emptyRuleSet)
    dummyClient
    (RuleSet.RuleSet [PureRule.AllowAll] [])
    4280

{{ Entry point: starts the npm registry proxy using {Npm.testConfig}. }}

Npm.main = do
  Server.startProxy Npm.testConfig Npm.parsePackageId
